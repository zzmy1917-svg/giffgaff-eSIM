## 大概要做什么

### 先看你的手机：

- 非国行 + 支持 eSIM：直接装 eSIM，最简单 
- 国行 / 不支持 eSIM（多数人）：下载`EasyEUICC APP`检测手机是否支持使用`小白卡`. 官网地址：[https://easyeuicc.org/zh-hans/](https://easyeuicc.org/zh-hans/)
- 把 eSIM 写进实体「小白卡」，插任何手机都能用 —— 本文主讲这条

### 需要准备：

- 一台安卓手机，请先用`EasyEUICC`检测是否支持`小白卡`
- 一张小白卡（电商约 20 元，比如：某宝480k 22 元）
- 一张国内单标 Visa / 万事达信用卡，卡上只标记 **`VISA`** 或 **`MasterCard`** ，无 **`银联`** 标记

### 如何保号？

#### 官方的保号条件是：180天必须有余额变动。发短信最划算。

> 这个卡在漫游时，拨打接听电话都是1英镑。接收短信免费，发送短信0.3英镑。**每年0.6英镑，15英镑可以使用25年**

#### 如何避免忘记发短信？

> 最好在手机日历中添加一个周期提醒。 比如添加两个生日提醒。
> 第一个生日设置为当天。第二个生日设置为180天后。提醒周期设置为每年重复，提前15天提醒。这样每半年提前15天就会有日历提醒。
